# Backend for PROD Hackathon

[![Java](https://img.shields.io/badge/Java-17-orange?logo=openjdk)](https://openjdk.org/)
[![Spring Boot](https://img.shields.io/badge/Spring%20Boot-3.5.6-brightgreen?logo=springboot)](https://spring.io/projects/spring-boot)

Бэкенд-сервис, написанный в одиночку для участия в хакатоне **PROD**.

## О проекте

Проект представляет собой полноценный бэкенд-сервис, реализующий всю необходимую логику для кейса хакатона.

**Ключевые факты о разработке:**
*   **Разработчик:** я писал в одиночку(кроме CI/CD и Docker, этим занимался мой сокамандник).
*   **Объем кода:** 2611 строк (без учета конфигурационных файлов по типу pom.xml).
*   **Срок реализации:** ~48 часов.
*   **Итоги хакатона:** **9 место** из 29 команд.
*   **Оценка бэкенда:** **7/10**.

## Стек

*   **Java 17**
*   **Spring Boot 3.2.0**
*   **JWT Auth**
*   **Swagger**
*   **Hibernate**
*   **База данных:** PostgreSQL
